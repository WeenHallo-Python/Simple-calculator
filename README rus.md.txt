README for Russian language
Здравствуйте уважаемый пользователь! Это простой калькулятор.
Этот калькулятор может складывать, вычитать, умножать, делить и возводить в степень.
Кроме того, в этом калькуляторе есть конвертор длины!
Если вам нужно преобразовать одну длину в другую, введите число в строке ввода и выберите, в какую единицу измерения перевести. После этого нажмите кнопку "="
Удачи!
